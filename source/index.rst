Welcome to the FT3-Module's documentation!
=============================================

FT3 is the recommended standards to handle accounts and tokens in the Chromia ecosystem. It allows dapps to make full use of Chromia Vault, including Single Sign-On (SSO), asset transfers and visibility on the Vaultâ€™s dapp explorer.

FT3 consists of a Rell module which contains blockchain logic, and client side lib which provides js API for interaction with the module.
